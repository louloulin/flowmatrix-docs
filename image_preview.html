Note: These SVG files need to be converted to PNG format before replacing the existing images in the project. You can use tools like ImageMagick, Inkscape, or online converters to perform this conversion.

Conversion Instructions

To convert these SVG files to PNG, you can use one of the following methods:

  1. Using ImageMagick: 
    convert new_header_image.svg images/header_image.png
    convert new_flowmatrix_arch.svg images/flowmatrix_arch.png
  2. Using Inkscape: 
    inkscape -z -w 1200 -h 400 new_header_image.svg -e images/header_image.png
    inkscape -z -w 800 -h 600 new_flowmatrix_arch.svg -e images/flowmatrix_arch.png
  3. Using an online converter:

    Visit a website like svgtopng.com and upload the SVG files.
